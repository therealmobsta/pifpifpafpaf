## Как опубликовать через GitHub Pages

1. Создать новый репозиторий на GitHub, например `magistracy-exam-trainer`.
2. Загрузить в корень репозитория все файлы из этой папки.
3. Открыть `Settings` -> `Pages`.
4. В блоке `Build and deployment` выбрать:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Нажать `Save`.

Через 1-2 минуты сайт будет доступен по ссылке вида:

```text
https://USERNAME.github.io/magistracy-exam-trainer/
```

## Как установить на телефон

Android: открыть ссылку в Chrome -> меню -> "Установить приложение" или "Добавить на главный экран".

iPhone: открыть ссылку в Safari -> "Поделиться" -> "На экран Домой".
